# Sintax Hotel

## Descripción del Proyecto

Sintax Hotel es un sistema de gestión hotelera completo desarrollado como proyecto grupal. La aplicación permite administrar de manera eficiente las operaciones diarias de un hotel, incluyendo gestión de reservas, habitaciones, clientes y servicios.

Este proyecto fue creado con tecnologías modernas de desarrollo web, priorizando la experiencia de usuario, rendimiento y escalabilidad.

## Características Principales

- Gestión integral de reservas hoteleras
- Administración de habitaciones y disponibilidad
- Panel de control intuitivo y responsive
- Sistema de autenticación de usuarios
- Interfaz moderna y amigable
- Validación de formularios en tiempo real
- Notificaciones y alertas personalizadas

## Requisitos Previos

Antes de comenzar, asegúrate de tener instalado:

- **Node.js** versión 20.19.0 o superior
- **npm** versión 8.0.0 o superior

## Instalación

Sigue estos pasos para configurar el proyecto localmente:

1. **Clonar el repositorio**
```bash
git clone https://github.com/Maxii34/ModuloN3-Front
```

2. **Navegar al directorio del proyecto**
```bash
cd front
```

3. **Instalar las dependencias**
```bash
npm install
```

4. **Iniciar el servidor de desarrollo**
```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:5173`


## Estructura del Proyecto

```
├── public/              # Archivos estáticos públicos

│

│   ├── assets/         # Recursos estáticos (imágenes, iconos)
│   ├── components/     # Componentes reutilizables
│   ├── pages/          # Páginas de la aplicación
│   ├── utils/          # Funciones auxiliares
│   ├── App.jsx         # Componente principal
│   ├── App.css         # Estilos del componente principal
│   ├── main.jsx        # Punto de entrada de la aplicación
│   └── index.css       # Estilos globales
│
├── .gitignore          # Archivos ignorados por Git
├── eslint.config.js    # Configuración de ESLint
├── index.html          # HTML principal
├── package.json        # Dependencias y scripts del proyecto
├── package-lock.json   # Versiones exactas de dependencias
├── vite.config.js      # Configuración de Vite
└── README.md           # Documentación del proyecto
```
